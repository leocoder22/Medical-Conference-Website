# Medical Conference Website

A full-stack conference management website developed using Python Flask for handling conference registrations, admin dashboard workflows, scheduling, and responsive event management.

## Features
- User registration system
- Admin dashboard
- Conference schedule management
- Responsive frontend UI
- Backend integration
- Authentication workflow

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLAlchemy
